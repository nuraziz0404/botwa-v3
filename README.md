# NfQ-BoT
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/nuraziz0404/botwa-v3
> cd Arya-Bot
> npm install
```
###### Run
```bash
> node index.js
```
